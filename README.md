# Нулевая смена

Полноценный браузерный аудио-хоррор и пошаговый тактический рогалик. Ты обслуживаешь аварийную станцию «Орфей-9», где часть радиосигналов настоящая, а часть имитирует людей — иногда твоим голосом.

## Что внутри

- пять сюжетных глав с отдельными картами и механиками;
- анализ настоящих и ложных сигналов по ритму, направлению и физическим следам;
- ремонт частотных контуров, клапаны давления и голосовые ключи;
- тёмная палуба с ограниченным сонаром;
- преследователь, которого можно запереть, оглушить, переждать или обойти;
- здоровье, кислород, энергия, давление и расходуемые инструменты;
- улучшения между главами, сохранение, контрольные точки и «Новая смена+»;
- три финала, включая скрытый полноценный финал.

## Управление

Работает мышь, сенсорный экран и клавиатура. `Tab` перемещает фокус, `Enter` или пробел активирует кнопку. Цифры `1–9` выбирают действие, `R` читает статус, `M` описывает карту, `S` включает сонар на тёмной палубе, `Escape` ставит игру на паузу.

Игра не требует зрения: все состояния и направления представлены текстом, кнопки имеют доступные имена, события передаются через ARIA live-регионы. Дополнительный голос браузера выключен по умолчанию, чтобы не спорить со скринридером.

## Проверка

Перед публикацией игра была полностью пройдена через Playwright от начала до скрытого финала. Отдельно проверены все реакции на преследователя, неправильные решения, смерти, сохранения, повреждённые сохранения, мобильный экран, горячие клавиши и попытки обойти обязательные задачи. Подробности — в [TESTING.md](TESTING.md).

Локальный запуск:

```bash
python3 -m http.server 4173
```

Воспроизводимый браузерный прогон:

```bash
npm install
npx playwright install chromium
npm run test:manual
```
